如何自訂 Brave 開啟的新分頁

由於Brave的 Dashboard 不能用 Vimium1，這當每次按t開啟新分頁時候，就無法享受到 Vimium 所帶來的爽感，所以想改成開啟搜尋引擎。

Brave把 homepage 設定藏很深
去 brave://settings 找（可以用搜尋的）：

  1. show home button -> set homepage - 設定主頁面
  2. new tab -> homepage - 將新分頁設定開啟主頁面

  1. 貌似是出於安全因素，有些Extenstions會進行攻擊https://www.reddit.com/r/brave_browser/comments/igxo25/vimium_on_braves_start_page/ ↩︎
